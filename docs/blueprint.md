# **App Name**: AsistenciaFacil

## Core Features:

- Creación de Grupos/Sesiones: Permite a los usuarios crear nuevos grupos o sesiones para gestionar la asistencia.
- Registro de Asistencia Individual: Permite marcar la asistencia de cada miembro del grupo o participante en la sesión.
- Visualización de Historial de Asistencia: Ofrece una vista del historial de asistencia para cada grupo o sesión.
- Generación de Informes de Resumen: Genera informes de resumen de la asistencia, incluyendo estadísticas y tendencias.

## Style Guidelines:

- Color primario: Verde suave (#A7D9A5) para reflejar crecimiento y positividad.
- Color de fondo: Blanco (#FFFFFF) para una apariencia limpia y clara.
- Color de acento: Un tono de verde más oscuro (#74B72E), para elementos interactivos y resaltar información importante.
- Fuente para cuerpo y encabezados: 'PT Sans', una fuente sans-serif que combina un aspecto moderno con un toque de calidez, adecuada tanto para encabezados como para el cuerpo del texto.
- Iconos claros y simples que representen acciones y estados relacionados con la asistencia.
- Diseño limpio y organizado con una navegación intuitiva para facilitar el registro y la consulta de la asistencia.
- Animaciones sutiles para dar retroalimentación visual al usuario al registrar la asistencia y al interactuar con la interfaz.
